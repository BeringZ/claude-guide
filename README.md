# Claude 完全指南 — 从入门到精通

一份精心编写的 Claude 使用指南，覆盖 **Claude.ai（网页版）** 和 **Claude Code（命令行）** 两种形态，从安装配置到高阶工作流的完整内容。

## 内容概览

| 章节 | 主题 | 内容 |
|------|------|------|
| **01** | 认识 Claude | 了解 Claude 是什么、能做什么、模型对比 |
| **02** | 安装与配置 | Claude Code 安装、登录、项目初始化 |
| **03** | 认识界面 | 网页版与 CLI 的布局和操作 |
| **04** | 程序架构 | 上下文机制、Skill 系统、MCP、工具调用 |
| **05** | 指令大全 | 内置命令、快捷键、提示词模板 |
| **06** | 使用技巧 | 五要素提问法、长文档阅读、写作流程 |
| **07** | 进阶工作流 | Projects、Artifacts、自定义 Skill、Hooks |

## 快速开始

直接打开 `index.html` 即可浏览完整指南。

```
open index.html
```

## 适合人群

- **初学者** — 从安装到第一个有效对话
- **日常用户** — 提升提问质量和输出效果
- **开发者** — 掌握 Claude Code 开发工作流
- **进阶用户** — 自定义 Skill 和自动化 Hooks
