# Claude 完全指南 — 从入门到精通

一份精心编写的 Claude 使用指南，覆盖 **Claude.ai（网页版）** 和 **Claude Code（命令行）** 两种形态，从安装配置到高阶工作流的完整内容。

在线浏览：[beringz.github.io/claude-guide](https://beringz.github.io/claude-guide/)

## 内容概览

| 章节 | 主题 | 内容亮点 |
|------|------|----------|
| **01** | 认识 Claude | Claude 是什么、发展简史、vs ChatGPT/Gemini、定价方案、模型对比 |
| **02** | 安装与配置 | 环境要求、三种安装方式、登录认证、项目初始化、多环境配置、问题排查、OpenClaw 代安装 |

| **04** | 程序架构 | 对话机制与上下文、Token 经济、Skill 系统、MCP、工具调用机制、权限系统、CLAUDE.md 最佳实践 |
| **05** | 指令大全 | 内置 Slash 命令、快捷键、Bash 命令速查、Git 工作流指令、提示词模板（通用 + 按角色分类） |

| **07** | 进阶工作流 | Projects、团队协作、Artifacts、自定义 Skill、Hooks、CI/CD 集成、7 天学习路径、完整工作流示例、真实案例 |

**共 7 章，50+ 个实用章节，30+ 条指令详解。**

## 快速开始

直接打开 `index.html` 即可浏览完整指南：

```
open index.html
```

或使用任意 HTTP 服务：

```
python3 -m http.server 8080
```

## 适合人群

- **初学者** — 从安装到第一个有效对话
- **日常用户** — 提升提问质量和输出效果
- **开发者** — 掌握 Claude Code 开发工作流
